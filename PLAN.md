### Milestone 16: 시트 구조 재설계 (단일 탭 · 회차 블록)

**목표:** 한 탭에 가로로 반복되는 회차 블록(6열 고정 + 1열 간격)을 파싱하고, 지정된 탭 하나만 노출하며, 오늘 날짜 이전 ~ 현재 회차까지만 렌더링한다.

**배경**
- 매니저 실제 시트는 한 탭에 1~5회차 시간표가 가로로 나열됨 (A~F 1회차, G 빈 열, H~M 2회차, ...)
- 학생은 오늘 날짜가 포함된 회차까지만 볼 수 있어야 함
- 다른 탭(시트) 정보는 학생에게 노출되지 않아야 함

**가정**
- 블록 크기는 **고정 6열** (시트 작성 실수로 깨지는 경우는 추후 대응)
- 블록 헤더 날짜 텍스트 형식은 기존과 동일 (`period` 텍스트 재사용)
- "회차 = 주차" — 네이밍은 **회차(session)** 로 통일

**Unit Tests (P0)**
- [x] 다중 블록 파싱: 2개 이상 블록이 빈 열(G, N, ...)을 경계로 독립된 회차로 분리하며 블록별 헤더 4필드(`programName`/`period`/`location`/`totalHours`) 개별 추출 → 01d9aaa
- [x] 미래 회차 필터: 오늘 기준 `period` 시작일이 미래인 블록은 결과에서 제외 → 94ae871
- [ ] 단일 탭만 페칭: 지정된 탭 외 Sheets API 호출 없음
- [ ] 구분 열 값 위반 감지: 구분 열에 `formattedValue`가 있으면 에러
- [ ] 구분 열 병합 위반 감지: 구분 열을 포함하며 폭이 2 이상인 병합이 있으면 에러

**Implementation**
